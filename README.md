# 2d-cooperative-control-of-two-arms-simple

## Описание проекта

Данный проект посвящён реализации кооперативного управления двумя плоскими роботами-манипуляторами в среде MuJoCo.

Цель проекта - разработать универсальный алгоритм, позволяющий двум манипуляторам совместно выполнять задачу переноса объекта с его передачей между роботами.

Алгоритм выполняет следующие этапы:

1. Первый манипулятор подходит к объекту.
2. Выполняется захват объекта первым манипулятором.
3. Объект переносится в промежуточную точку передачи и возвращяется на исходную точку.
4. Второй манипулятор подходит к объекту.
5. Второй манипулятор выполняет захват объекта.
7. Второй манипулятор переносит объект в конечную точку.
8. Объект освобождается в целевой области, после чего манипулятор возвращается в исходную конфигурацию.

Управление манипуляторами реализовано в виде последовательного алгоритма, состоящего из нескольких логически связанных этапов.

---

# Основные возможности

* Совместная работа двух манипуляторов.
* Захват, передача и перенос объекта.
* Управление на основе обратной кинематики.
* Последовательное выполнение операций.
* Плавное перемещение манипуляторов.
* Возврат манипуляторов в исходное положение после завершения задачи.

---

# Гибкость алгоритма

Одним из главных преимуществ проекта является его универсальность.

Алгоритм не привязан к фиксированным координатам. Для изменения сценария работы достаточно изменить координаты:

* стартового положения объекта;
* промежуточной точки передачи;
* конечной точки размещения объекта.

При этом логика работы алгоритма остаётся неизменной, а манипуляторы выполняют все этапы переноса в соответствии с новыми координатами.
